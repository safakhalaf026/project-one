## ğŸ§  About the Game
BUILD - A - WORD is a browser-based word game where players form as many words as possible using a randomized five-letter set. The game includes a timer, responsive UI, and animations such as confetti when the player finds all possible anagrams.


## âœ¨ Features
- Randomly selected letter sets
- Clickable on-screen keyboard 
- Timer starts only when the **Start** button is pressed  
- Words are validated once the **Enter** button is pressed 
- Letters can be cleared once the **Clear** button is pressed (represented by a cross)  
- Confetti animation when all anagrams are found  
- Displays how many anagrams remain  
- Pop-up modals for instructions and alerts  
- Mobile-responsive UI

## ğŸ¯ How to Play
1. Press **Start** to begin the timer 
2. Click the letter buttons to build a word 
3. Press the **Enter** to validate your word
4. Valid words appear in the list
5. Continue until all anagrams are found (a countdown of the words left will be present)
6. A confetti animation will play when you win
